# LMS BMQ POLINELA

Repositori ini berisi konfigurasi infrastruktur Docker untuk menjalankan sistem LMS tim AL-Banna Politeknik Negeri Lampung.

## Persiapan Awal
Pastikan laptop teman-teman sudah terinstall:
- [Docker Desktop](https://www.docker.com/products/docker-desktop/)
- [Git](https://git-scm.com/)

## Cara Menjalankan
1. Clone repositori ini:
   `git clone https://github.com/mrz464/lms_bmq_polinela.git`
2. Masuk ke folder proyek:
   `cd lms_bmq_polinela`
3. Buat file `.env` berdasarkan konfigurasi yang ditentukan.
4. Jalankan container:
   `docker compose up -d`
